# Backend Area

这是后端协作占位目录，用于明确前后端边界，避免后端实现散落到前端 UI 组件中。

## 后端同学负责

- API 路由与服务端接口
- 数据库连接、鉴权、旅行记录持久化
- AI/skill 调用、模型编排、异步任务
- 后端环境变量、部署配置、日志与监控

## 不要放在前端 UI 里的内容

- API key、token、数据库连接串
- 后端业务规则
- AI provider 调用细节
- 状态机迁移逻辑
- 真实用户数据处理逻辑

## 与前端对接方式

优先通过 `src/services/` 既有 service 边界对接，不要让 UI 组件直接调用后端 SDK。若需要新增字段或接口，请先在 `docs/TEAM_COLLABORATION.md` 记录变更意图，再由前后端共同确认。
